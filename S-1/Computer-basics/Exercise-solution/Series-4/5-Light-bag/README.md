# کیف سبک (Light Bag)

## 📝 صورت سوال

برنامه‌ای بنویسید که مبلغی را به تومان دریافت کند. با استفاده از سکه‌های ۱، ۵ و ۱۰ تومانی، کم‌ترین تعداد سکه مورد نیاز برای تشکیل آن مبلغ را پیدا و چاپ کنید.
(فرض کنید از هر سکه به تعداد نامحدود داریم.)

### ⚙️ محدودیت‌ها

  * **زمان:** ۱ ثانیه
  * **حافظه:** ۲۵۶ مگابایت
  * **محدوده اعداد:** $0 \le n \le 1000$

-----

## 📥 ورودی

ورودی شامل یک خط است که عدد صحیح غیرمنفی $n$ در آن آمده است.

## 📤 خروجی

کمترین تعداد سکه مورد نیاز را چاپ کنید.

-----

### 💡 نمونه‌ها

**ورودی نمونه ۱:**

```text
30
```

**خروجی نمونه ۱:**

```text
3
```

**ورودی نمونه ۲:**

```text
27
```

**خروجی نمونه ۲:**

```text
5
```

*(توضیح: ۲ سکه ۱۰ تومانی + ۱ سکه ۵ تومانی + ۲ سکه ۱ تومانی)*
