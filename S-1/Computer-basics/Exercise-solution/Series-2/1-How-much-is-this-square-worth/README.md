# این مربع چقدر می‌ارزه؟ (Determinant of a 2x2 Matrix)

## 📝 صورت سوال

برنامه‌ای بنویسید که دترمینان یک ماتریس $2 \times 2$ را محاسبه و چاپ کند.
دترمینان ماتریس $\begin{bmatrix} a & b \\ c & d \end{bmatrix}$ طبق فرمول زیر محاسبه می‌شود:
$$det = (a \times d) - (b \times c)$$

### ⚙️ محدودیت‌ها

  * **زمان:** ۱ ثانیه
  * **حافظه:** ۲۵۶ مگابایت
  * **محدوده اعداد:** $-10^4 \le a_{ij} \le 10^4$

-----

## 📥 ورودی

ورودی شامل دو خط است. در هر خط دو عدد صحیح آمده است که عناصر یک سطر از ماتریس را تشکیل می‌دهند.

## 📤 خروجی

دترمینان ماتریس داده‌شده را چاپ کنید.

-----

**ورودی نمونه ۱:**

```text

```

**خروجی نمونه ۱:**
