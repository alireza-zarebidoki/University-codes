# چی باقی مونده؟ (Remainder Without Modulo)

## 📝 صورت سوال

دو عدد صحیح $a$ و $b$ داده شده‌اند. برنامه‌ای بنویسید که باقی‌مانده‌ی تقسیم آن‌ها را **بدون استفاده از عملگر %** محاسبه کند.

### ⚙️ محدودیت‌ها

  * **زمان:** ۱ ثانیه
  * **حافظه:** ۲۵۶ مگابایت
  * **محدوده اعداد:** $-100 \le a \le 100$ و $1 \le b \le 100$

-----

## 📥 ورودی

در یک خط، دو عدد صحیح $a$ و $b$ با فاصله داده می‌شوند.

## 📤 خروجی

باقی‌مانده‌ی تقسیم $a$ بر $b$ را چاپ کنید.

-----

### 💡 نمونه‌ها

**ورودی نمونه ۱:**

```text
10 3
```

**خروجی نمونه ۱:**

```text
1
```

**ورودی نمونه ۲:**

```text
-10 3
```

**خروجی نمونه ۲:**

```text
-1
```

**ورودی نمونه ۳:**

```text
17 5
```

**خروجی نمونه ۳:**

```text
2
```
