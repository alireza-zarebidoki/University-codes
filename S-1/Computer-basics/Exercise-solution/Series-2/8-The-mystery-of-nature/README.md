# معمای طبیعت (Nature's Puzzle)

## 📝 صورت سوال

برنامه‌ای بنویسید که عدد $n$ را دریافت کرده و $n$ عدد اول دنباله فیبوناچی را چاپ کند.
**نکته:** دنباله فیبوناچی با دو عدد ۱ شروع می‌شود و هر عدد بعدی حاصل جمع دو عدد قبلی است ($1, 1, 2, 3, 5, \dots$).

### ⚙️ محدودیت‌ها

  * **زمان:** ۱ ثانیه
  * **حافظه:** ۲۵۶ مگابایت
  * **محدوده اعداد:** $1 \le n \le 50$

-----

## 📥 ورودی

در یک خط عدد صحیح $n$ داده می‌شود.

## 📤 خروجی

$n$ عدد اول دنباله فیبوناچی را با فاصله از هم در یک خط چاپ کنید.

-----

### 💡 نمونه‌ها

**ورودی نمونه ۱:**

```text
5
```

**خروجی نمونه ۱:**

```text
1 1 2 3 5
```

**ورودی نمونه ۲:**

```text
7
```

**خروجی نمونه ۲:**

```text
1 1 2 3 5 8 13
```

**ورودی نمونه ۳:**

```text
1
```

**خروجی نمونه ۳:**

```text
1
```
