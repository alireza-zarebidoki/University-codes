# ضرب اعداد موهومی (Complex Number Multiplication)

## 📝 صورت سوال

برنامه‌ای بنویسید که دو عدد مختلط را از کاربر دریافت کند، حاصل ضرب آنها را محاسبه کرده و نتیجه را نمایش دهد.
برای دو عدد مختلط به صورت $(a + bi)$ و $(c + di)$، حاصل ضرب به صورت زیر محاسبه می‌شود:
$$(a + bi) \times (c + di) = (a \times c - b \times d) + (a \times d + b \times c)i$$

که در آن:

  * قسمت حقیقی نتیجه = $(a \times c) - (b \times d)$
  * قسمت موهومی نتیجه = $(a \times d) + (b \times c)$

### ⚙️ محدودیت‌ها

  * **زمان:** ۱ ثانیه
  * **حافظه:** ۲۵۶ مگابایت

-----

## 📥 ورودی

برنامه باید چهار عدد حقیقی را از کاربر دریافت کند:

1.  قسمت حقیقی عدد اول ($a$)
2.  قسمت موهومی عدد اول ($b$)
3.  قسمت حقیقی عدد دوم ($c$)
4.  قسمت موهومی عدد دوم ($d$)

## 📤 خروجی

برنامه باید خروجی را دقیقاً به فرمت زیر چاپ کند:
`X + Yi`

که در آن:

  * $X$: قسمت حقیقی نتیجه با **دو رقم اعشار**
  * $Y$: قسمت موهومی نتیجه با **دو رقم اعشار**

-----

### 💡 نمونه‌ها

**ورودی نمونه ۱:**

```text
2.0 3.0
1.0 5.0
```

**خروجی نمونه ۱:**

```text
-13.00 + 13.00i
```
