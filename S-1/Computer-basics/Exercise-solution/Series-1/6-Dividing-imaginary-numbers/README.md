# تقسیم اعداد موهومی (Division of Complex Numbers)

## 📝 صورت سوال

برنامه‌ای بنویسید که دو عدد مختلط را از کاربر دریافت کند، حاصل تقسیم عدد اول بر عدد دوم را محاسبه کرده و نتیجه را نمایش دهد.
برای دو عدد مختلط به صورت $(a + bi)$ و $(c + di)$، حاصل تقسیم به صورت زیر محاسبه می‌شود:

$$\frac{a + bi}{c + di} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2}i$$

که در آن:

  * قسمت حقیقی نتیجه = $\frac{ac + bd}{c^2 + d^2}$
  * قسمت موهومی نتیجه = $\frac{bc - ad}{c^2 + d^2}i$

### ⚙️ محدودیت‌ها

  * **زمان:** ۱ ثانیه
  * **حافظه:** ۲۵۶ مگابایت

-----

## 📥 ورودی

برنامه باید چهار عدد حقیقی را از کاربر دریافت کند:

1.  قسمت حقیقی عدد اول ($a$)
2.  قسمت موهومی عدد اول ($b$)
3.  قسمت حقیقی عدد دوم ($c$)
4.  قسمت موهومی عدد دوم ($d$)

## 📤 خروجی

برنامه باید خروجی را دقیقاً به فرمت زیر چاپ کند:
`Quotient = X + Yi`

که در آن:

  * $X$: قسمت حقیقی نتیجه با **دو رقم اعشار**
  * $Y$: قسمت موهومی نتیجه با **دو رقم اعشار**

**نکته:** در صورتی که مخرج کسر صفر شود ($c=0$ و $d=0$)، برنامه باید پیغام `Division by zero is not allowed` را چاپ کند.

-----

### 💡 نمونه‌ها

**ورودی نمونه ۱:**

```text
2.0 3.0 4.0 2.0
```

**خروجی نمونه ۱:**

```text
Quotient = 0.54 + 0.31i
```

**ورودی نمونه ۲:**

```text
2.0 3.0 0.0 0.0
```

**خروجی نمونه ۲:**

```text
Division by zero is not allowed
```
